Tworzenie projektu w Altera Quartus dla DE-1

  1. Uruchom Altera Quartus II

  2. File -> New Project Wizard -> Next

  3. [Directory, Name...] Katalog roboczy na dysku J:\
  4. [Add Files] Next

  5. [Family & Device Settings]

    • Device family: Cyclone II

    • Available devices: EP2C20F484C7

    • Next
  6. [EDA Tool Settings] Next

  7. [Summary] sprawdzić czy Device zgadza się z 5. -> Finish

  8. W Project Navigator 2x LPM na Cyclone II: EP2C20F484C7

    • Device and Pin Options... -> Unused Pins -> Reserve all unused pins: As input tri-stated -> OK -> OK

  9. Assignments -> Import Assignments... -> wskazać DE1_pin_assignments.csv -> Open -> OK
